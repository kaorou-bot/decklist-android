# 🚀 开发者快速入口

> 使用 Claude Code 持续开发的快速入口

---

## ⚡ 30秒快速恢复

```bash
./quick_resume.sh
```

---

## 📚 必读文档（3个）

### 1️⃣ 整体状态
```bash
cat PROJECT_STATUS.md
```

### 2️⃣ 上次会话
```bash
cat SESSION_LOG.md
```

### 3️⃣ 当前任务
```bash
cat CURRENT_TASK.md
```
