## 🌙 深色模式优化

### 视觉改进
- **更深的背景色**: 使用纯黑色 (#121212) 替代深灰色，提升对比度
- **优化的文本颜色**: 主要文本使用纯白 (#FFFFFF)，次要文本使用浅灰 (#B3B3B3)
- **增强的按钮颜色**: 调整所有按钮颜色在深色模式下的可见度
- **改进的卡片背景**: 使用深灰色 (#1E1E1E) 与纯黑背景形成层次

### 主题配置优化
- 完整的 Material Design 3 颜色系统
- 改进的波纹效果（深色模式下使用白色半透明）
- 优化的状态栏和导航栏（深色背景，浅色图标）
- 更好的表面对比度

### 颜色对比度优化
- 主要文本: 从 #E6E1E5 提升到 #FFFFFF (更高对比)
- 次要文本: 从 #CAC4D0 调整到 #B3B3B3 (更好可读性)
- 提示文本: 从 #9E9E9E 调整到 #808080 (更柔和)
- 背景色: 从 #1C1B1F 改为 #121212 (更深的黑)
- 卡片背景: 从 #2B2930 改为 #1E1E1E (更好的层次感)

### 技术改进
- 添加了 `ripple_alpha` 颜色资源
- 改进的主题覆盖机制
- 完整的颜色语义化命名
- 更好的资源限定符使用

## 🚀 升级说明

### 从 v4.2.2 升级
- 直接安装新版本即可，无需卸载旧版本
- 深色模式设置会自动保留
- 所有数据会自动保留

### 深色模式切换
1. 打开应用
2. 点击右上角菜单
3. 选择"设置"
4. 在"主题"中选择：
   - 跟随系统（推荐）
   - 浅色模式
   - 深色模式
